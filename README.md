# Olá!

Boas vindas ao meu primeiro projeto :)

Este é o meu projeto para a _disciplina Programação FullStack_, uma das minhas matérias em meu primeiro semestre cursando _Ciência da Computação na FEI_.

Meu objetivo foi remeter a **estética da web retrô**, e tive como inspiração alguns sites que podem ser encontrados na indie web, como os disponíveis na plataforma neocites: https://neocities.org/

Dentro dos arquivos, há também as configurações base para rodar o site num servidor local, utilizando o Node.js.

Já tenho alguma experiência com HTML e CSS graças ao curso Técnico em Informática para Internet que fiz na Etec, por isso, não tive tanta dificuldade durante o desenvolvimento.
